Software Roundup

Throughout my website, there are many links to various software, teaching material, and tools that I have contributed too. For simplicity, here is a round up of these resources.

Software:

  • nQuack: An R package for predicting ploidy level from sequence data using site-based heterozygosity. (Also available on GitHub)
  • nQuack-AI: A data-based approach for predicting ploidy level from sequence data. Coming Soon! If you have sequence data with known ploidal level, please reach out!
  • AutoPop: R-based autopolyploid population simulations (Also available on GitHub)
  • gatoRs: Geographic and Taxonomic Occurrence R-Based Scrubbing R package (Also available on CRAN and GitHub)
  • Maximum likelihood based ancestral reconstruction. Available on GitHub: utremi & PaleoGenerate.
  • ridigbio: An R package to access iDigBio. (Also available on GitHub)
  • Biology PhD Stipends: A database and shiny app for Biology Ph.D. stipend benchmarking (Also available on GitHub)

Teaching Material:

  • Biodiversity Resource Round-up
  • Using Digitized Collections-Based Data in Research, Botany 2020, 2021, 2022, 2023, and 2025
  • Using R for Digitized Natural History Collections (dNHC) in Research, BCEENET 2021 Virtual Meeting
  • Introduction to R with Biodiversity Data, QUBES Educational Resources (Also available as a bookdown site and on GitHub)
  • Cleaning Biodiversity Data: A Botanical Example Using Excel or RStudio, QUBES Educational Resources
  • Course-based Undergraduate Research Experience: Florida Plants and Climate Change
  • API Office Hours with iDigBio API Working Group
  • Ecological Niche Modeling Crash Course
  • SPNHC-TDWG 2024, DemoCamp: An introduction to Geographic And Taxonomic Occurrence R-based Scrubbing (gatoRs): An R package and workflow for processing biodiversity data
